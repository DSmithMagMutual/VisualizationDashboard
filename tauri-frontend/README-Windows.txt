Jira Dashboard - Windows Installation Guide
============================================

QUICK START:
1. Double-click "app.exe" to run the application
2. No installation required!

FILES INCLUDED:
- app.exe (Main application)
- WebView2Loader.dll (Required dependency)

SYSTEM REQUIREMENTS:
- Windows 10 (version 1803+) or Windows 11
- 4GB RAM recommended
- Internet connection (for WebView2 runtime if needed)

TROUBLESHOOTING:
- If the app doesn't start, ensure you have the Microsoft Edge WebView2 Runtime
- Download from: https://developer.microsoft.com/en-us/microsoft-edge/webview2/

SUPPORT:
- For issues, check the application logs or contact support
